Where to begin

  • Workloads Start by building a set of workloads that represent your different applications.
  • Behchmarks Next, group your workloads into a benchmarks. A benchmark can have as many workloads as you wish to configure. You can use a workload multiple times.
  • Setup The setup prepares the enviornment to run the benchmark. This may include creating virtual machines, data stores, virtual networks etc.
